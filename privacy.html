개인정보 처리 안내

아이랑 성장가이드

이 앱은 서버 DB 없이 동작하는 정적 PWA입니다. 사용자가 입력한 아이 이름, 생년월일, 체크리스트 완료 여부는 현재 기기의 브라우저 저장소에만 저장됩니다.

저장되는 정보

  • 아이 이름 또는 별명
  • 아이 생년월일
  • 알림 사용 여부
  • 월령별 체크리스트 완료 여부

저장 위치

위 정보는 서버로 전송하지 않고 현재 기기의 localStorage에 저장됩니다. 브라우저 데이터 삭제, 앱 데이터 삭제, 기기 변경 시 기록이 사라질 수 있습니다.

방문 통계 분석

본 사이트는 방문 통계 분석을 위해 Google Tag Manager 및 Google Analytics를 사용할 수 있습니다. 이 과정에서 방문 페이지, 접속 환경, 이용 흐름 등의 정보가 수집될 수 있으며, 사용자가 입력한 아이 이름이나 생년월일은 분석 도구로 직접 전송하지 않습니다.

건강 정보 안내

앱의 발달·식사 정보는 참고용 가이드입니다. 진단이나 치료 목적이 아니며, 아이의 성장·발달·식사·알레르기와 관련해 걱정되는 부분이 있으면 소아청소년과에 상담하세요.

데이터 삭제

앱의 설정 화면에서 “데이터 초기화”를 누르면 현재 기기에 저장된 아이 정보와 체크리스트 기록이 삭제됩니다.
